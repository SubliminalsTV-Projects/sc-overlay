About
SC Blueprint Tracker is a desktop app that reads your Star Citizen game log to show the blueprint reward pool for the mission you're tracking, checking off each blueprint as you collect it. It runs as a lightweight in-game overlay — a transparent HUD that floats over the game and updates live while you play. Point it at your game.log below to get started, and optionally connect your subliminal.gg account to sync your collection and browse everything you've unlocked on the web. It can also use screen reading (OCR) — completely opt-in — to detect which mission you're tracking in-game.

Erkul Loadout URLs
Paste an erkul share URL per ship. The overlay reads the ship from each build automatically — when that ship spawns in-game, it switches to it. Click a row to make it the active overlay now.
Game Log
Pick the game.log file itself — the actual file, not the folder it's in. It lives inside your channel folder, e.g. …\StarCitizen\LIVE\game.log. Click Browse… to select the file, or Detect to auto-find your install and choose your channel (LIVE / PTU / EPTU / …).
subliminal.gg Sync
Get your token at subliminal.gg/blueprints → “Connect the desktop tracker”. Your collection and tracked mission then sync automatically.
When on, the app sends your Star Citizen Game.log to SubliminalsTV so we can find missions and blueprint drops the tracker gets wrong. Before it leaves your PC, your handle, account ID, IP address, and login session are stripped out, and chat is removed entirely — only mission and blueprint events remain. It's only ever used to improve the app. Needs a sync token above. Off by default.
Screen Reading (OCR)
Both use Windows' built-in text recognition and only ever run while Star Citizen is the focused window. Both are off by default — nothing on your screen is read unless you turn it on. (You can also flip these from the overlay's ⚙ menu.)
Reads the tracked-mission panel so the overlay knows which mission you've actually selected — ground truth the game log can't give (it sees every accepted mission the same). No account needed.
When you open a Fabrication Kiosk in-game, the app snaps the item's picture and contributes it to subliminal.gg's blueprint catalog — only items we don't have yet, nothing else on your screen. You're credited on the contributor leaderboard. Needs a sync token above.
Mining Assistant
A separate overlay window (tray → Mining Assistant) that reads the mining scanner's signature to name rocks, and the Refinement Center to alarm when a refine finishes. It's click-through until you hover it — just like the main overlay — so you won't click it by accident in-game. Voice, volume, alert-tone and announce controls all live in that window's ⚙.
Shows the window on its own when the scanner or Refinement Center screen is detected. If you close it, it stays out of the way for a bit before it can pop again.
Press this key in-game to show or hide the Mining Assistant window. Click the button and press a new key to change it — a single F3 / F13–F24, or any combo with Ctrl / Alt / Shift. Default is Shift+F3. Applies immediately, no restart.
In-Game Hotkeys
Do your hotkeys work when you alt-tab to the desktop, but do nothing while Star Citizen is focused? Windows is the reason: the game runs as administrator (Easy Anti-Cheat requires it), and another app's hotkeys can only reach it while it's focused if that app is also running as administrator.
✓ Running as administrator — your hotkeys work while the game is focused.
Relaunches the app with elevated permissions — you'll get a Windows UAC prompt, just click Yes. Nothing else changes; this only affects hotkeys while the game has focus. To skip this every launch, right-click the app's shortcut → Properties → Compatibility → “Run this program as an administrator.”
Binding Chart Overlay
Pick a PNG (transparency supported — e.g. your joystick binding chart) and press the hotkey in-game to toggle it full-screen over gameplay (click-through, won't interfere). To change the hotkey, click the button and press your combination — include Ctrl, Alt, or Shift, or a single F3 / F13–F24 key on its own (these are free in Star Citizen; F4–F12 are already bound). Default is Alt+F3. Applies immediately, no restart.
Overlay Performance
Press this key in-game to toggle the whole blueprint overlay on/off. Click the button and press a new key to change it — a single F3 / F13–F24, or any combo with Ctrl / Alt / Shift. Default is F3. Applies immediately, no restart.
Turn this off if the overlay crashes your game. Blueprint tracking and collection sync keep running without it — you just lose the on-screen panel. Takes effect immediately, no restart. Tip: the binding-chart hotkey (Alt+F3) opens another overlay window, so leave that off too while testing.
On an AMD GPU and the game crashes (Vulkan "device lost") when the overlay is up? Turn this on. It forces the overlay fully off the GPU and switches to a simplified, no-blur panel — which stops that crash while keeping the overlay visible. Slightly plainer look, a little more CPU. Requires restarting the app.
Off by default. The overlay renders on the CPU, which avoids crashes some GPUs (especially AMD) hit when the overlay runs over the game. Turn this on only if you have GPU headroom and don't get crashes. Changing this requires restarting the app.
