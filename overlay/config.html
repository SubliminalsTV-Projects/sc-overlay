Setup & defaults the essentials, sensible out of the box
Game log
iPick the game.log FILE itself (not its folder) — e.g. …\StarCitizen\LIVE\game.log. Browse… to select it, or Detect to auto-find your install and channel (LIVE / PTU / …).
Appearance
iRestyle the overlay to a Star Citizen manufacturer. Mobiglas is the clean default; Drake is rugged burnt-orange; Anvil is military teal-green; Grey's is grungy CRT green; Auto matches the ship you're flying. Applies immediately. (Per-widget sizing: drag a widget's corner in arrange mode.)
Auto (matches the ship you're flying) and Mobiglas are free. 🔒 Pinning one manufacturer regardless of ship is a subscriber perk — pick one to preview it, then it reverts. Subscribe any way below and it stays unlocked for as long as you're subscribed: Twitch → SubClub → subliminal.gg →
Hotkeys
iClick a button then press your combo. Use Ctrl/Alt/Shift + a key, or a bare F3 / F13–F24 (those are free in Star Citizen; F4–F12 are taken). Applies immediately, no restart.

Performance & fixes
Turn off if the overlay crashes your game — tracking + sync keep running without it.
iOn an AMD GPU and the game crashes (Vulkan “device lost”) when the overlay is up? Turn this on — it forces the overlay off the GPU with a simpler no-blur panel. Requires restarting the app.
iOff by default — the overlay renders on the CPU, avoiding crashes some GPUs hit. Turn on only if you have GPU headroom and no crashes. Requires restarting the app.
Hotkeys work on the desktop but not in-game? iDo your hotkeys work on the desktop but not while Star Citizen is focused? The game runs as admin (anti-cheat), so another app's hotkeys only reach it while focused if that app is ALSO admin.
✓ Running as administrator — hotkeys work while the game is focused.
Relaunches elevated (a Windows UAC prompt — click Yes). Only affects hotkeys while the game is focused. Tip: shortcut → Properties → Compatibility → Run as administrator to skip this each launch.
iIf a widget ends up off-screen (e.g. dragged onto another monitor), this returns every widget to its default on-screen spot.

About
SC Overlay is a lightweight, transparent in-game HUD for Star Citizen. It reads your game.log to show the blueprint reward pool for the mission you're tracking and checks off each blueprint as you collect it — plus a Mining Assistant (signature scanner + refinery-job timers) and manufacturer skins that automatically match the ship you're flying. Opt-in screen reading (OCR) can detect your tracked mission in-game and help crowdsource item images for the catalog. Point it at your game.log below to get started, and optionally connect your subliminal.gg account to sync your collection and browse everything you've unlocked on the web.
